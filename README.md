# ShortURL
***วิธีการติดตั้ง***

> จะมี directory ที่เป็นส่วนของ Front-end และ Back-end แยกกัน  
> จำเป็นต้องติดตั้ง package ในทั้งสอง directory   


## **Front-end**

เปิด teminal ใหม่  
cd .\Front-end\  
npm i  
npm run dev  


> run ที่ port 5173


## **Back-end**

เปิด teminal ใหม่  
cd .\Back-end\  
npm i  
npm run back  

   

> run ที่ port 8080
